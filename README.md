## Brief Overview
My version of the "ZettBrett" board, designed as a part of the Advanced Digital Electronics Certification.

Software used for the Schematics/PCB: **KiCAD 8.0**

## System Blocks
- AMD/Xilinx Zynq-7000 SoC
- 1GB of DDR3 memory
- Gigabit Ethernet
- USB 2.0 HS OTG
- 4GB eMMC memory
